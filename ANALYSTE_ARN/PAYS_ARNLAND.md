## TENSIONS STRATÉGIQUES POUR SCÉNARIOS

### 1. Diaspora mercurienne (vulnérabilité principale)
- 8% de la population (3.4M) = justification officielle d'Olamao pour intervenir
- Concentration géographique (Grand Est + PACA/Occitanie) → zones d'influence et de recrutement
- TANTALE utilise ces zones comme terrain d'opération

### 2. Dépendance gazière
- 63% du gaz via pipeline contrôlé par Bothnia + Mercure
- Coupure = crise économique immédiate
- LNG alternatif disponible mais limité

### 3. Corruption systémique
- 119/180 TI → fragilité institutionnelle
- Judiciaire peu fonctionnel : aucune récupération d'actifs à ce jour
- 50% de recettes fiscales non perçues

### 4. Polarisation politique
- APP (pro-OTAN/EU) vs ASP (critique, soutenu narrativement par Mercure)
- Faible confiance dans les institutions nationales (64% défiance)
- L'ASP amplifie la rhétorique pro-SCO de Mercure dans l'opposition

### 5. TANTALE + N.O.M
- TANTALE : proxy militaire de Mercure, actif surtout dans les zones à forte diaspora
- N.O.M : groupe terroriste fictif, anti-OTAN, anti-capitaliste, couverture environnementale

### 6. Position géostratégique
- Accès atlantique + méditerranéen unique dans le Skolkan
- La Garonne rejoint HBordeaux (port mercurien) → point de friction potentiel
- Frontière avec Mercure : zone Mont Blanc
