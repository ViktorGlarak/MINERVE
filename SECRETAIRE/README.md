# SECRÉTAIRE — Mistral NeMo 12B

## Rôle
Expert en rédaction française, communication professionnelle et synthèse de documents.

## Modèle Ollama
`mistral-nemo:latest` (7.1 GB) — `ollama run mistral-nemo`

## Domaines de compétence
- Rédaction de mails professionnels
- Synthèses et résumés de documents
- Reformulation et correction de textes
- Rapports, comptes-rendus, présentations
- Communication en français soigné

## Quand l'appeler
- Rédiger ou améliorer un mail
- Résumer un document long
- Reformuler un texte technique pour un public non-technique
- Préparer un compte-rendu de réunion
- Corriger l'orthographe et le style

## Fichiers
- `MEMOIRE.md` — Préférences de style, contexte métier récurrent, destinataires fréquents
- `MODELES.md` — Templates de documents réutilisables
- `SESSIONS/` — Logs des sessions de rédaction
