- **Publier / statut** : « Publier » **ne change plus** le statut (simple sélection) ; le bouton **« 🗂 Extraction complète »** produit désormais le **`.json` de session** (identique à l'export MELMIL) **+ commit** (publiés → ✅ Envoyé + dépubliés/republiables). Porté sur les 3 instances (collab 7BB + 2 vierges).

- **Diffusion par LOTS INCRÉMENTAUX** (solution finale, [[ARCH-009]]) : MASTER écrit `moteur/actus/lot_<stamp>.js` (push remplace-par-id) + régénère le manifeste `actus_index.js` ; le joueur **ajoute** chaque lot sans écraser les anciens, fichiers/zip restent petits.
