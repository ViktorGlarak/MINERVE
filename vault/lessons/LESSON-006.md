# LESSON-006 — Langue du tweet = langue de l'avatar

## Symptôme observé
Tweets rédigés dans la mauvaise langue pour l'avatar (07.05.12Bi/Ci corrigés le 02/06).

## Correctif / règle à appliquer
Vérifier la langue dans `tweets_data.js`. Ex. Tikhanov/Saniki = **EN** ; @HmunikVoice/@clambroise55/@VoixDACia/@TemoignageDAC/@MaiaKovalenko = **FR**.
