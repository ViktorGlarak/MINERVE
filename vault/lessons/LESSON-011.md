# LESSON-011 — Règles de prompt LTX 2.3

## Correctif / règle à appliquer
- Format : **paragraphe unique fluide**, verbes au présent, 4-8 phrases.
- Décrire **uniquement le mouvement** (le contenu visuel est déjà dans l'image source).
- **ia2v** : la synchro labiale doit TOUJOURS être dans le prompt.
- Logos / texte précis → **overlay Premiere Pro**, jamais via LTX.
