# 📚 Bothnia — économie & énergie

## Économie
Économie de marché très prospère : **PIB ~296 Mds $**, /hab ~48 770 $, croissance +2,5 %, monnaie **Botmark**, tax/PIB 32,7 %. 90 % privé ; **BIH** = bras d'investissement de l'État (participations Telecom, Econombank) ; **Botoil** = pétrole d'État (monopole raffinage/import).

## Énergie (levier)
Importe ~**30 % de son gaz de Mercure** (pipeline central Skolkan) + production North Sea / **champ gazier BGroningen** ; **sert de conduit gazier vers l'Arnland**. Nucléaire jusqu'à **2033** puis renouvelables (36 000 MW éolien). Ressources : gaz/pétrole North Sea, lignite Bannova-Rheinland, métaux Belluxa.
