# 📚 Arnland — infrastructure

**National Infrastructure Plan** (aligné UE) ; double façade **Atlantique + Méditerranée**.
- **Numérique** : cyber via ENISA + **NATO CCDCOE** ; **DESI dernier (29e/28)** ; **5G Dushman** (contrat 2028, en examen sécuritaire) → point de friction ; nœuds vitaux **HLyon & HPARIS** ; Arnland Telephone Network (monopole jusqu'à 2027).
- **Énergie** : **EDA** (lié au nucléaire **Dushman**) ; nucléaire 71,7 % / renouvelable 21,3 % / fossile 7 %. ⚠ **Gaz 63 % via pipeline Skolkan (Bothnia/Mercure)** + LNG (HDunkerque/HMontoir/HFos). Barrages CNI.
- **Points critiques** : pipeline gazier, HPARIS/HLyon (numérique), **HMARSEILLE** (3e port Médit., 16 % diaspora MER).
