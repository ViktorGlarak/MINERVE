# 📚 Mercure — relations internationales & infrastructure

## Relations
| Partenaire | Nature |
|---|---|
| **Arnland / DR** ([[ENT-arnland]]) | **Hostile** — rival principal ; diaspora 8 %, gaz, TANTALE, R2P |
| **Dushman** | Partenaire stratégique — équipements, 5G, ports (HSete), ferroviaire |

## Infrastructure
**Gaz** : réseau 18 600 km → **Nord Fluss** (Bothnia + Nord Arnland + export via Den Helder) / **Sud Fluss** (Sud Arnland + Framland + GNL Gênes). Réserves ≈ 90× la conso. **Transports** : **Cenari Base Tunnel**, aéroports HMunik (hub fret Dushman) + Milan, opérateur **Skolkavia**. Nucléaire ≈ 31 % électricité.
