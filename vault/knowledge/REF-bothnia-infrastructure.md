# 📚 Bothnia — infrastructure

Plan **NEIDP** (4 piliers : hub commercial Skolkan · économie numérique · sécurité énergétique · croissance régionale).
- **Maritime** : **BRotterdam** (plus grand port Skolkan, 12ᵉ mondial) · BCuxhaven · Ben Helder ; flotte marchande 750 navires. ⚠ **~25 % du pays sous le niveau de la mer ; ~50 % du PIB en zone inondable** → digues vitales.
- **Rail** : Bothnian National Rail ; hubs Bobrus/BFrankfurt/Bannova → réseaux ARN/MER (pertinent pour le soutien FGF-L 7BB).
- **Air** : aéroport **Brahea/Schiphol** (2ᵉ Skolkan). **Route** : >30 ponts sur le Rhin inférieur.
- **Numérique** : **Bobrus = plus grand hub internet du Skolkan** ; investissement **Dushman** ; 5G.
